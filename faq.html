How to change font in Samsung、OnePlus、Color OS System phone?

Most Samsung、OnePlus、Color OS System have font manager, you can install the font APP,and go to the font manager screen to choose the new font.

how to change font in Huawei and Xiaomi phone?

This phone have theme manager, Fonter will pack the font to a Theme package. then go to the Theme manager to pick the new theme.

How to change font on other system?

Fonter need root permission to change font file, after that need restart.

How to change app font?

Change app font need Xposed framework (http://www.xposed.info), after install and enable Xposed framework, enable Fonter module to use this feature.

How to use custom font?

On the settings screen, you can set where to put custom font files.

How to change system font on Samsung 4.3+ system?

On Samsung 4.3+ system, you only can set the font that installed from the Samsung app store. So, most custom font not work on Samsung 4.3 system. you need follow those steps to change system font(NEED ROOT):
1.Go to font display settings screen set your current font is a default one (that is, one that comes with the phone)
2. Using Fonter to set system font
3. Reboot your device

Can not restore to factory font using the Restore function?

The restore function can only restore the font when you first using Fonter to change system font. We call this as the SYSTEM DEFAULT FONT not means SYSTEM FACTORY FONT.

How to build font app using the custom ttf files

There is an app to do this thing, Font Style Maker for FlipFont ( https://play.google.com/store/apps/details?id=com.fontsfree.apkmodder) This app will allow you to create FlipFont compatible Font apps right on your device. All you need is a ttf font file from the internet or create your own from one of the many font creators available. Then place the file on your device or email it to yourself and use this ttf apk converter app to create a custom font app with your new font. This Custom Font Maker is for Samsung Galaxy devices designed to be compatible with Monotype Imaging Inc.'s FlipFont® program and will install new free fonts on your Samsung Galaxy that are compatible with the FlipFont® program on your phone.
