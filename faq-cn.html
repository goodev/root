* 如何获取ROOT权限

1. 先试试 https://towelroot.com/tr.apk ，也可以到中国各大市场搜索“神奇ROOT”， 建议直接使用 https://towelroot.com/tr.apk ， 市场上的汉化版有些是盗版，里面带有很多广告。
2. 如果 https://towelroot.com/ 无法获取ROOT，可以试试QQ的 http://root.qq.com/  和 360 的 http://root.360.cn/

注意： 不推荐使用除了 360 和 QQ root 之外的一键root工具来获取ROOT， 经过测试发现 其他国产ROOT工具大部分带有大量的广告和偷偷摸摸安装垃圾应用的行为，所以不推荐使用。

建议： 在您使用各种一键ROOT工具获取到ROOT后, 安装 SuperSU 来管理ROOT权限， 在SuperSU 成功安装并获取到 ROOT 权限后，把其他一键ROOT工具和ROOT管理工具都卸载掉。 SuperSu 下载地址：
https://play.google.com/store/apps/details?id=eu.chainfire.supersu 不能上 play 市场的可以到百度下载： http://shouji.baidu.com/soft/item?docid=7127403﻿

一、为啥需要ROOT才能换字体？

ROOT 权限是安卓手机的最高权限，一个应用一旦有了 ROOT 权限，就可以对手机系统为所欲为了，所以 ROOT 权限是很危险的、但是也是很强大的。对于初次使用安卓手机的用户，建议不要授予那些来历不明的应用 ROOT 权限，否则会 造成一些后果；轻则 通讯录、短信、通话记录被盗取；重则 偷偷发短信、偷偷订阅一些收费服务 造成财产损失。
除了一些定制的安卓系统（三星手机、Color OS 手机、小米手机、华为手机 等）支持免 ROOT 替换系统字体；其他大部分基于谷歌原生安卓系统的手机都需要替换系统的 字体文件（位于 /system/fonts/ 目录下） 才能更换系统 字体， 所以在这些系统上 Fonter 需要 ROOT 权限才能修改系统字体。
Fonter 承诺： 只用 ROOT 权限用户修改替换系统字体相关文件，不做其他用户不知情的任何操作。

二、三星手机如何换字体？

大部分三星手机都集成了字体管理功能，如果您在系统设置中发现有“字体样式”相关选项，则说明您的手机带有字体管理功能。 带有该功能的手机，可以直接安装下载的 字体 APK(安卓应用安装包)文件， 安装完字体后，在系统的“字体样式” 设置中启用安装的字体即可更换字体，无需 ROOT 权限。
注意： 部分 Color OS 和一加手机也集成了字体管理功能，也可以使用该模式来更换系统字体。

三、华为手机如何换字体？

华为手机带有主题功能，Fonter 把字体制作为华为的主题包，然后到主题商店界面中选择新生成的字体包来设置。操作步骤如下:
“设置”-“全部设置”-“显示”-“字体样式”,在里面找到要替换的字体，然后点击应用按钮重启后即可生效。

四、魅族手机如何获取 ROOT 权限？

魅族系统需要先登录魅族账户，然后在账户里选择“开放系统权限”选项来开启 ROOT 功能。

五、小米手机如何换字体？

小米手机带有主题功能，Fonter 会把字体制作为主题包，然后在主题界面选择混搭来修改字体，具体操作步骤如下：
主题界面选择‘分类’；再选择最下面的 ‘混搭’；然后选择‘混搭’界面中的‘字体’功能；最后在‘字体’界面中的‘我的’下应用该字体
注意 该功能仅支持部分小米系统。

六、为何替换字体后，部分文字还是原来的样子？

安卓系统保护多种字体，有 Robot、SansSerif、Arial 等，还包含多种样式： 粗体、斜体、粗斜体。大部分安卓应用都使用通用的 Robot 字体，所以在替换字体的时候，我们只替换了 常用的普通 Robot 字体。 所以有些界面用到了其他字体，则不会修改。 另外在替换的字体文件中，可能字库不全，缺少的文字也会用系统字体来显示。

七、如何恢复为原来的字体？

设置界面有个 恢复默认字体 功能，点击该功能来恢复原来的字体。

八、温馨提示，换字体必读

由于安卓系统的各种定制版本存在，所以有些版本目前还没测试过，可能存在使用 Root 模式替换字体后，系统无法启动的情况。因此： 当使用 Root 换字体模式时候，请务必先备份手机资料，防止字体替换失败导致手机无法开机，资料丢失。

九、如何修改单个应用的字体？

使用侧边栏里面的 “修改应用字体” 功能来换应用的字体。该功能依赖 Xposed 框架（http://www.xposed.info），需要先安装 Xposed 框架并启用 Fonter 模块，然后就可以使用 “修改应用字体” 功能来给单个应用换字体了。
注意： Xposed 框架只支持 4.0.3(包含) 以上的安卓系统，如果您的手机无法开启 Xposed 框架，则无法使用 “修改应用字体” 功能。

十、如何使用 TTF 字体文件？

在设置中可以设置自定义字体目录，你可以把您的 TTF 字体文件放到该目录下，这样“设置应用字体”的时候会查找自定义的字体文件。
